# 📚 Mangachan

**Mangachan** is a modern and minimal manga reading web application built with cutting-edge web technologies. It lets users **discover**, **search**, and **read** manga series through a clean, responsive, and user-friendly interface.

---

## 🚀 Features

- 🔍 **Manga Search** – Instantly search for your favorite manga titles.
- 📖 **Reader View** – Seamless, paginated reading experience with smooth navigation.
- 🌙 **Dark/Light Mode** – Toggle themes for comfortable day or night reading.
- ⚡ **Fast and Lightweight** – Built with performance and speed in mind.
- 📱 **Responsive Design** – Works flawlessly across desktops, tablets, and smartphones.

---

## 🛠 Tech Stack

- [**Next.js**](https://nextjs.org/) – React framework for fast SSR and SSG.
- [**TypeScript**](https://www.typescriptlang.org/) – Strongly typed JavaScript for safer, scalable code.
- [**Tailwind CSS**](https://tailwindcss.com/) – Utility-first CSS framework for rapid UI development.
- [**Shadcn UI**](https://ui.shadcn.com/) – Accessible and beautifully designed component library.
- [**next-themes**](https://github.com/pacocoursey/next-themes) – Theme toggling made easy.
- [**MangaDex API**](https://api.mangadex.org/docs/) – Source of up-to-date manga content.

## 📦 Getting Started

### Prerequisites

- [Bun](https://bun.sh/) – Fast all-in-one JavaScript runtime
- Node.js 18+ (optional, only for compatibility with some tools)

### Installation

```bash
# Clone the repository
git clone https://github.com/rujesh01/mangachan.git

# Navigate to the project folder
